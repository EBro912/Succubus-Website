SauceNAO Commands

Sauce An Image

Usage

/sauce [url]

Description

Search the SauceNAO databases for the given image url. If no image url is provided, the first image in the past 20 messages will be used.

Note

Valid image formats include .PNG, .JPG, and .GIF

Set Result Count

Usage

/sauce-setresults amount

Description

Modify the amount of results the sauce command returns, from 1 to 5.

Required Permissions

Manage Messages

Lock Sauce Command

Usage

/sauce-lock channel

Description

Locks the sauce command to the specified channel, so that sauce queries may only be run in that channel.

Required Permissions

Manage Channels

Unlock Sauce Command

Usage

/sauce-unlock

Description

Unlocks the sauce command if it is locked.

Required Permissions

Manage Channels
